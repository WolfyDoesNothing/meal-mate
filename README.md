MealMate

A mobile app for meal planning and recipe management, developed using **Java** and **Firebase**.

Features
- Create, edit, and delete recipes  
- Store and sync data in **Firebase Firestore**  
- Share shopping lists via **SMS**  
- Shake detection for random meal suggestions  
- Clean UI using **Material Design**

Tech Stack
Java | Android Studio | Firebase | Firestore | XML | Material Design

How to Run
1. Clone the repository  
2. Open with **Android Studio**  
3. Connect to Firebase (if needed)  
4. Run the app on an emulator or Android device  

This project was built as part of university course and demonstrates user experience design and Firebase integration.
